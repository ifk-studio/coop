что бы преобразовать js в валидный json необходимо
- удалить 1 и последню строки
- удалить в первой непустой строке все символы до {
- открыть в vim и провести замены по следующим регэкспам
%s/[^"]\([{]*\)\([a-zA-Z1-90_]\+\)\s*:/\1"\2":/g   - обернуть все name: в кавычки "name":
%s/null/""/g
%s/,\n\s\+\([}\]]\)/\1/g - замена ,} или ,] на }} }]
%s/\([+-]\d\+\)/"\1"/g  - замена для примера +100 на "+100"
- удалить все коментарии
- валидность можно проверить на jsonlint.com
